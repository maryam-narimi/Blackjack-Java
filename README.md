#  Simple Blackjack Game in Java

A simple **text-based Blackjack game** written in Java.  
Play against the computer (dealer) directly in the console!

---

##  Game Overview

Blackjack is a popular card game where the goal is to get as close to **21** as possible without exceeding it.  
In this project:

- You play as the **player** against the **dealer** (computer).  
- The dealer draws cards until reaching **17 or more**.  
- **Aces** are counted as 11 or 1 automatically.  
- You can choose to **Hit** (take a card) or **Stand** (stop taking cards).  

---

##  How to Run

1. **Compile the Java code:**
   
javac Blackjack.java

```bash
Run the game:

java Blackjack

Follow the on-screen prompts:

Enter h or hit → to take another card (Hit)

Enter s or stand → to stop taking cards (Stand)

After each round, you can choose to play again.```
